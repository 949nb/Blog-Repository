Git工作流程

workplace(工作区)=>index/staged(暂存区)=>repository(本地仓库)=>remote(远端仓库)

git 有三种状态：

  1. 已提交(committed)
  2. 已修改(modified)
  3. 已暂存(staged)

git.config一般在三个地方

系统级 用户级别 和 项目级别

git add

git commit

git status

撤销操作

git reset –soft HEAD^ 撤销前一次commit

取消暂存区中的某个文件 git reset HEAD

查看自己远程仓库 地址

git remote -v

添加一个远程仓库

git remote add

设置完仓库之后可以通过命令 拉去仓库中的代码

git fetch

创建分支

git branch
